====== Postfix ограничить число исходящих соединений ======

При использовании Postfix в качестве relay, все исходящие письма отправляются через почтовик провайдера, который в свою очередь ограничивает число паралельных входящих соединений.

Чтобы ограничить количество паралельных исходящих соединений, делаем следующее:

<code>
# /etc/postfix/master.cf

# Если не используете relay, то можно просто ограничить количество исходящих подключений
smtp      unix  -       -       n       -       -       smtp

# При использовании relay достаточно изменить только эту строчку (вместо "-" ставим "1"). Цифра 1 = количество соединений
relay     unix  -       -       n       -       1       smtp
</code>

Также добавим строчек в main.cf
<code>
# /etc/postfix/main.cf

# При использовании relay достаточно добавить только этот параметр
relay_destination_concurrency_limit = 1

# Если не используете relay, то
smtp_destination_concurrency_limit = 1
</code>
